
В качестве входов используются A0 для контроля общего напряжения,
A1-A3, A6, A7 для контроля отдельных напряжений.

Выходы для светодиодов: 8-12 для светодиодов отдельных напряжений, 13
--- для светодиода общего напряжения.

Когда общее напряжение становится ниже V_TOTAL_LIMIT, светодиод
начинает мигать и включается сигнализация с частотой TONE_FREQ_TOTAL.

Когда отдельное напряжение становится ниже VOLT_LIMIT, соответствующий
светодиод начинает мигать и включается сигнализация с частотой
TONE_FREQ_SINGLE.

Когда отдельное напряжение становится ниже среднего на VOLT_THRES, на
индикаторе соответствующий аккумулятор помечается знаком "0",
светодиод начинает мигать и включатся сигнализация с частотой
TONE_FREQ_SINGLE.
